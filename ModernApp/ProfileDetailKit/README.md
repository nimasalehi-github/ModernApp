# ProfileDetailKit

A description of this package.
یک پکیج جدید به اسم ProfileDetailKit برات می‌سازم که فاز B+C (جزئیات، ویرایش، ذخیره، حذف، افزودن) رو پوشش بده. همه چیز با async/await، @MainActor، @Published، SwiftUI و Concurrency-safe خواهد بود و کاملاً bilingual با شماره‌گذاری.
